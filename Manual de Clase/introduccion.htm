INTRODUCCIÓN

Este juego consiste en ir formando una palabra más larga que el contrincante el máximo de letras de las que se dispone para hacerlo son 9, las cuales el jugador las va pidiendo al juego, ya sea vocal o consonante.

Tiene otro panel que es la parte de cifras, consiste en alcanzar el número exacto o aproximado del objetivo que el juego nos da, si te acercas será una puntuación menor a la del exacto esto se irá comprobando con el contrincante, para ver quien de los dos suma puntos.

Este juego es un ejercicio de la asignatura Desarrollo de Interfaces, del curso 2º DAM, hemos puesto en práctica todo lo aprendido durante el curso, hemos trabajado con los tipos de layouts, funcionalidades para la parte lógica de la aplicación, esta muy bien porque es una forma de aprender cosas nuevas y poder realizar proyectos por tu cuenta.

Han surgido complicaciones a la hora de su elaboración, pero el resultado es exitoso, más adelante se podrían agregar algunas funcionalidades nuevas para la mejora de este proyecto. Este trabajo está hecho con java swing, concretamente en Eclipse, Java.
